Programa de Busqueda de Series y Películas


Como proyecto final del curso de Python, nos pusimos el objetivo de crear un programa donde tengamos el acceso rápido y seguro de los próximos estrenos de películas. Por esto creamos MyStreamerSearcher, ya que este programa nos ayudará a encontrar lo que buscabamos, de una forma sencilla e intuitiva.
Todo el código fue creado en Python con los conocimientos previos adquiridos en el curso, usamos librerias como Colorama para el estilo de la app y Json para los datos.
Utilizamos funciones, bucles, excepciones, opción de autenticación, listas y diccionarios.
Nuestro programa le va a permitir:
- Iniciar sesión con nombre de usuario y contraseña.
- Ver TODOS los estrenos del mes (mediante una lista de diccionarios).
- Buscar dentro de la lista de películas según diferentes filtros (en este caso genero y plataforma de streaming).
- Generar una lista de películas favoritas, poder ver dicha lista, agregar y eliminar contenido de la misma.

Python Colorama JSON

Análisis de Covid (Biogenesys)

El propósito del proyecto se basa en el análisis estratégico de datos con el fin de entender y ayudar a la empresa Biogenesys a expandirse en latinoamérica.

Con el fin de entender este análisis con mayor profundidad, hemos realizado un dashboard interactivo mediante el cual podremos elegir que datos explorar.
